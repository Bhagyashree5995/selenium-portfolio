# 🧪 Selenium Automation Portfolio

A professional test automation portfolio built with **Python**, **Selenium WebDriver**, and **pytest**, following the **Page Object Model (POM)** design pattern.

---

## 👩‍💻 About This Portfolio

This portfolio demonstrates my skills as a QA Engineer transitioning into test automation. It covers end-to-end UI testing of real web applications using industry-standard tools and practices.

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| Python 3.x | Programming language |
| Selenium WebDriver | Browser automation |
| pytest | Test framework |
| Page Object Model | Design pattern |
| GitHub Actions | CI/CD pipeline |
